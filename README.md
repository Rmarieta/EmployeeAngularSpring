# Employee Management System Using Angular, Spring Boot and MySQL

Application containing both backend and frontend components providing the possibility to host data about employees. It is possible to :
- add employees
- edit their details
- delete employees
- search for employees through their name, email, phone number or job title

All changes are persistent to the database. The frontend is coded using Angular and TypeScript, the backend using Spring Boot and Java.

To run the backend, run the server/ app in IntelliJ
To run the frontend, run the following command in client/ :
```
ng serve
```
The app will be running on <code>http://localhost:4200</code>
